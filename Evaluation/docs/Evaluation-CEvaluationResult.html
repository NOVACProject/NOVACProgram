namespace Evaluation

class CEvaluationResult

CEvaluationResult is a generic class to describe the results after evaluating a single spectrum. It contains an array of CReferenceFitResult:s - one for each reference spectrum that was included in the fit, and other information about the fit - such as Delta, ChiSquare, the number of iterations required and the fitted polynomial.
