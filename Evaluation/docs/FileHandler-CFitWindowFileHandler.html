public RETURN_CODE ReadFitWindow(Evaluation::CFitWindow &window, const CString &fileName, int index = 0)

Reads the desired fit window from the file.

Parameters:

fileName - - the name and path of the file to read from
index - - the zero-based index of the fit window to read(each .nfw file can contain several fit-windows)

Returns:
