class CConfigurationSetting::WindSpeedMeasurementSetting

Attribute Summary
public bool
automaticWindMeasurements
True if this system is supposed to make wind speed measurements automatically
public double
desiredAngle
The desired angle [degrees] between the two measurement directions OR the desired distance [meters] between the two measurements at plume-height.
public int
duration
The preferred duration of each measurement [in seconds]
public int
interval
The preferred interval between each measurement [in seconds]
public double
maxAngle
Measurements will only be performed if the centre of the plume is within +-maxAngle from zenith
public double
minPeakColumn
Measurements will only be performed if the peak-column(minus the offset) is larger than 'minPeakColumn' ppmm
public int
stablePeriod
Measurements will only be performed if the centre of the plume is relatively stable over the last 'stablePeriod' number of scans
public int
useCalculatedPlumeParameters
This is true if we should trust and use the values for wind-direction and plume-height calculated by the program
